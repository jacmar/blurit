# Configurazione dell'app Python Selective Focus su GitHub Pages Per far funzionare l'app con PyScript su GitHub Pages, segui questi passaggi: ## 1. Struttura dei file Crea una struttura di file così: ``` /repository index.html (HTML principale con PyScript) manifest.json (Manifest PWA) sw.js (Service Worker) /icons icon-72x72.png icon-96x96.png ...altri file icona... ``` ## 2. Utilizzo di PyScript PyScript è un framework che permette di eseguire Python nel browser. Il file index.html che ti ho fornito include tutti i riferimenti necessari a PyScript. Funziona così: - Carica automaticamente le librerie Python come NumPy e PIL - Esegue il codice Python direttamente nel browser - Converte i file immagine in un formato compatibile con Python ## 3. Note importanti: ### Prima esecuzione La prima volta che apri l'app, PyScript scaricherà l'ambiente Python (~15-20MB). Questo potrebbe richiedere un po' di tempo, ma avviene solo la prima volta e poi viene memorizzato nella cache. ### Compatibilità - Funziona meglio su Chrome, Firefox e Safari desktop - Su dispositivi mobili potrebbe essere più lento, specialmente la prima volta ## 4. Limitazioni - Le prestazioni di elaborazione delle immagini dipenderanno dalle capacità del dispositivo - L'elaborazione di immagini multiple richiederà più memoria e potrebbe essere lenta su dispositivi meno potenti
